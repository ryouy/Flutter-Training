## Getting Started

To explore or build the projects in this repository:

1. **Install Flutter SDK** (latest stable version)
   Follow the official installation guide to set up Flutter on your platform.

2. **Clone the repository**

   ```bash
   git clone https://github.com/ryouy/Flutter-Training.git
   cd Flutter-Training
   ```

3. **Open in your IDE**
   Use Visual Studio Code, Android Studio, or another supported editor.

4. **Run a sample app**

   ```bash
   flutter pub get
   flutter run
   ```

   Ensure a device or emulator is connected.
